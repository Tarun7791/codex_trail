This project is a **working front-end demo** of an accountability platform for students and developers who struggle with daily consistency.

## Quick start

```bash
python3 -m http.server 4173
```

Open: `http://localhost:4173`

## What to demo (2-minute walkthrough)

1. Click **Try Live Demo**.
2. Add a goal like `Solve 3 DSA problems`.
3. Mark a goal as **Done** and watch XP/consistency update.
4. Add a progress post in the feed.
5. Click **Simulate a Productive Day** to auto-complete goals and advance streak.
6. Toggle between dark/light theme.
